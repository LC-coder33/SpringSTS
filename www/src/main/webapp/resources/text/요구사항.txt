계급별 서비스 제공. 비회원, 회원, 관리자는 각자의 고유의 기능이 있고 이 기능들만 사용 가능하다.

로그인 기능
아이디와 비밀번호를 입력. 비밀번호 텍스트 칸은 암호화 되어있다. 만약 틀리면 아이디 혹은 비밀번호가 잘못되었다며 다시 입력하길 요구한다.
모든 비밀번호(로그인, 회원가입, 관리자 로그인)는 암호화 처리되어 보여지지 않는다.

회원가입 기능
아이디 중복을 먼저 검사한다. 중복이 아님이 확인이 되면 회원가입을 완료 시킬 버튼이 생성 되며 비밀번호 및 이름 입력 칸이 나타난다.
비밀번호는 정규식을 사용하여 8~16자에 숫자 문자 특수문자 모두 포함해야만 한다.
회원가입이 끝나면 팝업이 나오고, 이를 닫으면 바로 메인화면으로 복귀하게 된다.

비회원 조회 기능
비회원은 도서를 살 수는 없지만 구매 가능 도서들을 체크할 수는 있다. 그러나 해당 도서를 구매하려 한다면 로그인을 요구하는 주의사항이
화면에 나타난다.

관리자 로그인 기능
관리자 코드만 입력하면 관리자 로그인이 된다. 이후, 관리자는 도서를 종이책 / ebook 프레임을 띄워서 각자 관리 가능하다. 
각각 입고, 삭제, 수정이 가능하다.
프레임에 열거되어있는 레이블에 해당하는 정보를 옆의 텍스트에 입력 한 뒤 입고를 누르면 도서가 입고가 된다.
원하는 도서를 목록에서 클릭 후 수정을 누르면 수정하기 위한 정보를 입력받을 프레임이 새로 나온다. 
이 프레임에 정보를 입력한 뒤 수정하기를 누르면 수정이 완료되고 테이블에 수정 된 정보가 포함 된 열들이 다시 나열된다.
리스트에 삭제를 원하는 항목을 클릭한 뒤 삭제버튼을 누르면 해당 도서는 삭제가 된다.

회원 로그인 기능
종이책: 종이책과 이북 모두 검색 기능이 있다. 검색한 글자를 포함한 제목을 가진 도서가 리스트로 뜬다.
종이책은 가운데 리스트로 나타난다. 가격 및 재고가 보이는데, 만약 구매자가 재고 이상으로 개수를 설정한 뒤 구매하려 하면
메세지가 뜨면서 진행이 안된다.
구매 총 가격은 구매*구매개수로 계산되며 해당 회원의 구매내역에 들어가면 총 가격을 볼 수 있다.
