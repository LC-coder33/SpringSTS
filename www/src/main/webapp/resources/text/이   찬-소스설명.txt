* 제목 : 상품 구매 장바구니 UI

* 구현한 기능
1. 상품 정보를 입력하여 주문하는 기능
	- 상품 이름, 개수, 개수 당 가격, 배송비를 입력하여 주문
	- 위의 항목 중 하나라도 빠지면 다시 입력하라는 알람창을 띄움
	- 이후 주문하기 버튼을 누르면 장바구니 추가
	- 장바구니가 비어있다는 문구 삭제
2. 장바구니에 추가된 물품의 가격에 대한 자동 계산
	- 장바구니에 올라간 물품들의 상품금액 및 배송비는 자동으로 계산됨
	- table의 tbody cart__list에 열을 추가하여 상품에 대한 정보가 입력 됨
	- 주문금액에는 테이블의 본문에 있는 상품들 전체의 상품금액과 배송비의 총합계가 입력 됨
3. 장바구니의 물품 삭제
	- 장바구니에 있는 물품을 선택하여(다중 선택 가능) 삭제 가능
	- 상품이 삭제되면 해당 상품의 금액 만큼 상품금액 및 배송비가 차감되고 총 주문금액 또한 차감
4. 장바구니 전체 체크 기능
	- 헤더의 상품정보행 왼쪽의 allChk체크박스행을 클릭하면 목록 전체를 선택 가능

* 구현 과정 풀이
  상품 구매 화면이 로딩 될 때, 선택상품금액, 총 배송비, 주문금액이 입력되는 DOM 요소의 초기 값 설정한다. 해당 요소들을 미리
0이라는 문자열로 설정해서 NaN(not a number)에러 발생을 방지한다. 이후, 해당 요소에 추가되는 값들을 전부 parseInt를 통해
숫자로 변환시켜 계산에 용이하게 하였다.
  이후, 테이블의 본문의 id인 cart__list를 carList변수에, 장바구니가 비어있음을 알리는 텍스트는 hidden변수에 저장하여 전역변수 선언을
함으로써, 행이 추가되거나 비워질 때를 비롯해 어느 상황에서도 해당 요소들을 사용하기 용이하게 설계하였다.

  addRow() 함수를 통해 테이블 본문에 행을 추가하는 기능을 추가하였다. 상품 이름, 개수, 개수 당 가격, 배송비를 입력받고 주문하기
버튼을 누르면 장바구니가 비어있다는 문구가 없어지고 하나의 행으로 만들어져 추가되는 방식이다. 위 4가지 요소 중 하나라도 빈 칸인 채로 
버튼을 누르면 정보 입력 후 다시 주문해달라는 알람창이 출력된다. 이후, 개수와 가격을 곱하여 상품 금액을 출력하고 배송비 또한 
배송비 칸에 출력된다. 주문금액 또한 장바구니에 행이 추가됨과 동시에 업데이트된다. 또한, 해당 행이 추가되면서 맨 앞 열에 체크박스를
추가하는데, 이를 통해 다음에 설명할 delRow()함수를 이용할 수 있다.

  delRow() 함수를 통해 테이블 본문에 저장 된 행을 삭제하는 기능을 추가하였다. 행의 가장 앞 열인 체크박스를 체크 후 선택상품삭제
버튼을 누르면 해당 행은 사라지고 이에 해당하는 가격 또한 상품 금액, 배송비, 주문금액에서 차감된다. 여러 행을 체크한 후 삭제 가능하다.

  allChk1() 함수를 통해 체크박스 전체 체크하는 기능을 추가하였다. 헤더의 첫 열인 allChk를 클릭하면 테이블의 모든 열이 체크된다.
delRow() 함수와 연계하여 주문목록을 편하게 지우는데 사용할 수 있다.


* 주요 사용 기술
  순수 자바 기술만을 사용. DOM조작을 통하여 동적 조작을 하였다. 
  DOM조작을 통해 동적 조작, 즉 이벤트 핸들링은 onclick을 통해서만 함수(function)를 사용하는 방식으로 하였다.
  addRow(), delRow(), allChk1() 함수를 onclick으로 동작시켜 이벤트 핸들링을 하였다. 
  parseInt를 통한 숫자 계산, for문을 통한 열 순회 및 if문을 통한 체크박스 체크 여부 확인을 을 통해 주요 기능들을 완성시켰다.
  .style.display로 요소의 숨김 및 표시를 제어하였다.
  페이지 로드 시 초기값을 설정하여 NaN을 방지하였다.


	- document.getElementById()로 입력된 상품정보, 수량, 가격, 배송비를 가져왔다.
	- insertRow(), insertCell()로 새로운 행과 열을 추가하였다.
	- parseInt()로 문자열을 숫자로 변환하여 숫자를 계산하였다.
	- innerHTML로 계산된 값들을 화면에 표시(addRow())하였다.

	- for문으로 장바구니 테이블의 모든 행을 순회했다.
	- cells[0].children[0].checked로 체크박스 상태 확인하였다.
	- parseInt()로 선택된 상품의 가격 계산하였다.
	- deleteRow()로 선택된 행을 삭제하였다.
	- innerHTML로 총액을 업데이트(delRow())하였다.
  
	- if문으로 모든 행의 체크박스의 체크 상태를 확인하였다.
	- for문으로 모든 행의 체크박스를 check 상태로 변경하였다.
